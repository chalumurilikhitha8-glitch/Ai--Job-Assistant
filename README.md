# 🤖 AI Job Application Assistant

An AI-powered web application that helps users **analyze resumes, match job descriptions, and improve job applications** using smart automation and NLP concepts.

---

## 🚀 Live Demo

https://ai--job-assistant-dejnds9zrugaicztaurjty.streamlit.app/

---

## 🧠 Overview

This tool compares a **resume with a job description** and provides insights such as:

* Match score (ATS-style)
* Missing keywords
* Suggestions to improve resume

It is designed to help job seekers improve their chances of getting shortlisted.

---

## ✨ Features

* 📄 Resume analysis
* 🎯 Job description matching
* 📊 ATS score calculation
* 🧠 Smart suggestions
* ⚡ Fast and simple UI (Streamlit)

---

## 🛠️ Tech Stack

* Python
* Streamlit
* NLP (text processing)
* TF-IDF / keyword matching
* Scikit-learn (if used)

---

## ⚙️ How It Works

1. User pastes resume
2. User pastes job description
3. Text is cleaned and processed
4. System compares keywords and similarity
5. Outputs:

   * Match score
   * Missing skills
   * Improvement suggestions
